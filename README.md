# Learning From Data – 2024-2025 Fall

This repository contains homework assignments and the term project developed for the **Learning from Data** course during the **2024–2025 Fall semester** at **Istanbul Technical University (ITU)**, Faculty of Computer and Informatics.

## 📚 Course Overview

The course focuses on fundamental machine learning concepts, including:

- Supervised learning
- Linear models
- Bias–variance tradeoff
- Model evaluation
- Regularization techniques
- Feature engineering
- Optimization methods
- Generalization theory

## 📂 Repository Contents

- Homework implementations
- Experimental studies
- Model evaluations and analysis
- Term project source code
- Reports (if applicable)

## 🛠️ Technologies Used

Depending on the assignment:

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook
